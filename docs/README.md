# PigTex Desktop Docs Index

This public repo keeps only desktop documentation that is safe to publish.

## Included

- [PIGTEX_TRUST_POLICY.md](./PIGTEX_TRUST_POLICY.md)

## Excluded From The Public Tree

- Internal release runbooks
- Incident and go-live checklists
- Product backlogs and execution packs
- Private partner or production integration specs

## Placement rule

- Keep only documents that are safe for public readers and useful for contributors.
- Internal operating material should live outside the open-source repository.
